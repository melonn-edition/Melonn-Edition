Nuevas funcionalidades de API Melonn

Disponible en Melonn Colombia Melonn México

Una nueva capa de integración para automatizar tu operación con más control, trazabilidad y escalabilidad.

API V2 te permite gestionar más procesos desde tu sistema y automatizar actualizaciones mediante webhooks. Incluye órdenes de ingreso, devoluciones, webhooks para estados de órdenes de venta y ajustes de inventario, además de rutas más ordenadas y una documentación más clara.

Migración requerida

Si tu operación hoy usa API V1, debes migrar a V2 antes de noviembre de 2026. A partir de esa fecha, la versión anterior dejará de estar disponible. Melonn acompañará el proceso de transición.
